这个 Demo 验证了什么

对应规格书 (Photo-Aiid-system) 的 V1.0 范围：链接照片 → AI 自动打标 → EXIF 提取 → 动态索引表 → 模板化批量重命名 → CSV 导出。布局采用规格书验证过的「左侧设置面板 + 顶部 Tab + 瀑布流画廊」。

与正式版的差异

  • 识别引擎：支持两种——① Claude 视觉 API（仅在电脑网页版 claude.ai 的 Artifact 内可用，手机 App 暂不支持该桥接）；② Ollama 本地推理（把本文件下载到 Mac，用 Chrome 打开，左侧切换引擎），后者已经符合规格书「零云端」原则。正式版按规格迁移至 ONNX Runtime + CLIP。
  • Ollama 跨域设置：终端执行 launchctl setenv OLLAMA_ORIGINS "*" 后重启 Ollama；模型需支持视觉（如 gemma3、llava、qwen-vl 系列）。
  • 重命名：浏览器无法改写本地文件，Demo 提供「下载重命名副本 ZIP」模拟物理重命名；正式版 (Electron) 可直接操作文件系统。
  • 语义搜索：Demo 把标签索引发给模型做匹配；正式版应使用本地向量库 (SQLite + Vector Extension) 做 embedding 检索。
  • 缓存：Demo 刷新即失；正式版须持久化扫描结果。
