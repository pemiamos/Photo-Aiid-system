# Photo-Aiid-system v3.1.2

AI 照片管理 + 网页征稿协作系统 —— 大图 / HEIC 解码修复版（macOS / Apple Silicon · Windows x64）

## ✨ 这个版本修了什么

v3.1.2 在 v3.1 系列基础上，重点解决**部分照片无缩略图、且无法做 AI 检测**的问题，并让版本号可在界面直接确认。

### 🖼️ 超大图 / HEIC 照片不再「无缩略图、无 AI」⭐ 重点
此前以下两类照片会卡在「待分析」、画廊显示蓝色「?」占位、AI 也跳过不处理：

- **上亿像素的大图**：Pillow 默认有约 8900 万像素的解码上限（防「解压炸弹」），超过就解码报错。本工具处理的是本地可信照片，已**解除该上限**，超大图正常生成缩略图与参与分析。
- **HEIC / HEIF（iPhone 默认格式）**：此前后端缺少解码插件，PIL **根本打不开 `.heic`**。现已内置 `pillow-heif` 解码，HEIC 照片可正常显示缩略图并做 AI 识别。

附带优化：大尺寸 JPEG 改为**按目标尺寸降采样解码**（`draft`），省内存、更快，避免大图卡死。

### 🧹 跳过 macOS 伴随文件（`._xxx`）
- 在非 Mac 格式磁盘（exFAT/FAT 外置盘等）上，macOS 会为每个文件生成一个 `._` 开头的 **AppleDouble 伴随文件**，无图像内容。此前它们也被当成 `.jpg` 收录，卡在「待分析」、显示「?」占位、污染照片总数。
- 现在**扫描时直接跳过**所有隐藏文件与 `._` 伴随文件；并在每次扫描时**自动清理历史已误入库**的这类记录。

### 🔖 版本徽章自动显示真实版本号
- 顶部徽章此前**写死为 `v3.1`**，新旧版本看着一样、无法区分。现改为自动读取版本号，安装后徽章会显示 **v3.1.2**，一眼即可确认是否更新成功。

### 📦 含 v3.1.1 的全部修复
- **画廊解除 200 张上限**，改为渐进式渲染，上万张大库完整展示、滚动更顺
- **分析批次异常隔离**：单张出错不再拖垮整批，进度条贴合实际进度、不再提前消失
- **字段拖移修复**：侧栏命名字段 / 投稿管理书籍栏拖拽排序更跟手

> 本地照片整理（AI 识别、批量重命名、语义搜索、离线地理编码、拖拽到桌面等）与网页征稿、云端投稿管理、OSS/R2 双云存储**全部保留**。

## 📦 下载与安装

提供 **macOS（Apple Silicon）** 与 **Windows（x64）** 两个平台，按需下载。

### macOS（Apple Silicon M1/M2/M3/M4）
下载 `Photo-Aiid-system_3.1.2_aarch64.dmg`：

1. 双击 `.dmg`，将 `Photo-Aiid-system.app` 拖入「应用程序」文件夹
2. **首次打开会被 macOS 拦截**（提示“无法验证开发者”或“已损坏，无法打开”）——这是因为本应用未经苹果签名，**不是病毒**
3. 解决方法（任选一种）：
   - **右键点击 App 图标 → 打开 → 在弹窗中再次点击「打开」**（只需一次，之后可正常双击）
   - 或在终端执行：
     ```bash
     xattr -dr com.apple.quarantine "/Applications/Photo-Aiid-system.app"
     ```

### Windows（x64）
下载 `Photo-Aiid-system_3.1.2_x64-setup.exe`（NSIS 安装包，推荐）双击运行安装；另提供 `Photo-Aiid-system_3.1.2_x64_zh-CN.msi` 供企业部署。首次运行若被 SmartScreen 拦截，点「更多信息 → 仍要运行」即可（未签名所致，非病毒）。
