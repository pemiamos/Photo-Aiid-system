# Photo-Aiid-system v3.3.0

AI 照片管理 + 网页征稿协作系统 —— 双版本模式 + 缩略图稳定性 + OpenAI 引擎（macOS / Apple Silicon · Windows x64）

## ✨ 这个版本新增了什么

v3.3.0 最大的变化是把 App 拆成**两种使用模式**，普通用户和组稿摄影师各取所需；同时修好了征稿看板缩略图的稳定性，新增 OpenAI 识别引擎与一批使用体验细节。

### 🎚️ 双版本模式：普通用户版 / 高级摄影师版 ⭐ 重点
同一个 App，顶部一键切换，重启自动记住上次选择。

- **🙂 普通用户版（默认启动）**：界面只保留「画廊 / 索引表 / 使用教程」与扫描、AI 分析、重命名等本地整理功能，**隐藏所有投稿/征稿入口**（投稿管理、投稿模式、云端征稿）。给「只想整理自己照片」的人一个干净、零概念负担的界面。
- **📷 高级摄影师版**：点右上角「切换到高级摄影师版」，当前会话立刻解锁完整功能——投稿管理、投稿模式、网页征稿、云端协作全部回来。给画册组稿、多人征集的团队用。
- 实时切换、互不打断；从高级版切回普通版时会自动退出投稿模式、不会卡在已隐藏的页面。

### 🖼️ 征稿看板缩略图：本地缓存 + 方向 / 完整度 / HEIC 修复
- **本地缓存**：缩略图首次访问由服务器抓一次、落盘缓存，之后从本地直发，彻底解决「时好时坏、只刷出一部分」；省流不变。
- **修正方向**：手机竖拍照片（横向像素 + EXIF 旋转标记）此前方向/取景错乱，现按 EXIF 自动转正。
- **完整显示**：从「裁切填满方格」改为「等比缩放完整可见」，宽幅/竖幅照片不再被切成中间一条。
- **支持 HEIC**：手机直出的 HEIC 转 JPG 后也能出缩略图；个别处理不了的会优雅占位、不再黑块。

### 🤖 新增 OpenAI 识别引擎
- 识别引擎新增 **OpenAI API**，支持自定义 API 地址（兼容代理/网关）。
- 模型下拉**实时从接口拉取真实可用列表**（自建/代理网关各不相同），并自动过滤掉非视觉模型；可「自定义…」手填任意模型名。

### 📂 画廊 / 历史索引体验增强
- **显示图片位置 + 打开所在文件夹**：每张照片卡片顶栏显示其所在目录，右上角「📂 打开文件夹」一键在 Finder / 资源管理器中定位该文件。
- **尺寸与大小常显**：文件名下方始终显示像素尺寸与文件大小，不再只在分析成功时才出现。
- **历史索引点图放大**：历史索引搜索结果支持点击放大（左右切换 / Esc 关闭）。
- **投稿管理记住上次书目**：重开自动回到上次选中的那本书。

> 本地照片整理（AI 识别、批量重命名、语义搜索、离线地理编码、拖拽到桌面等）与网页征稿、云端投稿管理、OSS/R2 双云存储**全部保留**。

## 📦 下载与安装

提供 **macOS（Apple Silicon）** 与 **Windows（x64）** 两个平台，按需下载。

### macOS（Apple Silicon M1/M2/M3/M4）
下载 `Photo-Aiid-system_3.3.0_aarch64.dmg`：

1. 双击 `.dmg`，将 `Photo-Aiid-system.app` 拖入「应用程序」文件夹
2. **首次打开会被 macOS 拦截**（提示“无法验证开发者”或“已损坏，无法打开”）——这是因为本应用未经苹果签名，**不是病毒**
3. 解决方法（任选一种）：
   - **右键点击 App 图标 → 打开 → 在弹窗中再次点击「打开」**（只需一次，之后可正常双击）
   - 或在终端执行：
     ```bash
     xattr -dr com.apple.quarantine "/Applications/Photo-Aiid-system.app"
     ```

### Windows（x64）
下载 `Photo-Aiid-system_3.3.0_x64-setup.exe`（NSIS 安装包，推荐）双击运行安装；另提供 `Photo-Aiid-system_3.3.0_x64_zh-CN.msi` 供企业部署。首次运行若被 SmartScreen 拦截，点「更多信息 → 仍要运行」即可（未签名所致，非病毒）。

> 升级提示：本版的缩略图本地缓存 / 方向修复 / HEIC 支持在**征稿服务器**侧；OpenAI 引擎、双版本模式、打开所在文件夹等在**本机后端与前端**。如自建征稿服务器，请同步更新服务器上的 `backend/` 并重启 `photo-intake` 服务（`bash scripts/sync-backend.sh`）。
