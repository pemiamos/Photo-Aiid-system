# Photo-Aiid-system v3.1.0

AI 照片管理 + 网页征稿协作系统 —— 从「单文件夹临时整理」升级为「多文件夹持久库 + 全局检索」（macOS / Apple Silicon · Windows x64）

## ✨ 这个版本带来了什么

v3.0 打通了本地整理与云端征稿；**v3.1 把本地整理侧从「一次只看一个文件夹」升级成「持久化的多文件夹照片库」，并新增跨全库的全局检索**。

### 🗂️ 多文件夹持久索引库
- **换文件夹不再清空历史**：每浏览/扫描一个文件夹就并入库，历史文件夹的元数据与 AI 标注**长期保留**；只要路径不变、已分析过的随时可被检索到
- **缩略图改为按需生成**：扫描阶段不再全量生成缩略图，首次被查看时才实时生成并缓存 —— 上万张规模也能轻量持有，磁盘占用极小
- **已索引文件夹注册表**：后端提供文件夹列表/移除/清空/清理失效路径的接口

### 🔍 索引表「全局索引」模式
- 索引表新增 **🔍 全局索引** 一键切换：**跨所有已索引文件夹**按关键词检索（摄影师 / 地点 / 标签 / 类别 / 文件名）
- 搜索框下方两行快捷标签：**最近搜索关键词** + **全库最高频关键词**
- 结果以**画廊同款瀑布卡片**展示，缩略图懒加载 + 分页「加载更多」，扛得住大库
- 三个操作按钮（全局索引 / 执行物理重命名 / 导出索引 CSV）同行并贴页面右缘

### 🖱️ 拖文件夹进画廊
- 直接把文件夹拖入画廊区即自动扫描并展示（桌面 App 可获取真实路径）

### 🛠️ 体验修复
- **摄影师识别**：原文件名中的署名（如 `张扬的小强-南京.jpg`）现优先于 AI 猜测，并支持更长昵称；相机自动文件名前缀（IMG/DSC/DJI 等）不再被误判为摄影师
- **重命名预览即时刷新**：在侧栏调整命名字段后，画廊文件名预览立即按新模板更新，并新增手动「刷新」按钮
- **库统计修复**：「已分析 / 标签数」此前因字段名错误恒为 0，现已正确统计
- **全局分析**可对已分析照片重复重新分析（带确认）
- **投稿管理 · 书籍栏**：超长列表竖向滚动、项目块拖拽排序、底部加「星尘远征队」署名
- **征稿地址**：粘贴整段投稿页/二维码链接（含 `?code=`）不再导致投稿码误判无效，并自动回填投稿码

> 本地照片整理（AI 识别、批量重命名、语义搜索、离线地理编码、拖拽到桌面等）与 v3.0 的网页征稿、云端投稿管理、OSS/R2 双云存储**全部保留**。

## 📦 下载与安装

提供 **macOS（Apple Silicon）** 与 **Windows（x64）** 两个平台，安装方式与 v3.0 一致，详见 README。
