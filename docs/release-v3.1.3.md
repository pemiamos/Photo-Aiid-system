# Photo-Aiid-system v3.1.3

AI 照片管理 + 网页征稿协作系统 —— 保留原始文件名、增强历史索引检索（macOS / Apple Silicon · Windows x64）

## ✨ 这个版本修了什么

### 🔎 原始文件名永久保留，历史索引检索更全 ⭐ 重点
此前**物理重命名后，原文件名会被新名覆盖、不再保存在任何地方**。如果原名里带有 AI 看不出来的信息（如地名「九龙潭」），重命名后就再也搜不到了。

- 新增 **原始文件名**字段：首次扫描时记录，**物理重命名永不覆盖**。原名里的地名、人名等信息长期保留。
- **历史索引/全局搜索现在同时检索**：当前文件名 + **原始文件名** + 类别 + 标签 + 描述 + **地点** + **摄影师**。
  - 之前搜索只覆盖「文件名/类别/标签/描述」，**遗漏了地点与摄影师字段**；现已补全，与历史索引的设计一致。
  - 这样即便照片被重命名为 `杨建发_自然风光_20210626.jpg`，搜「九龙潭」仍能命中。
- **老库自动迁移**：升级后首次启动会自动为已有照片回填原始文件名（此时尚未物理重命名的，原名即当前文件名，信息完整保留）。

> 建议：若你打算批量物理重命名，**升级到 v3.1.3 后再做**，即可确保原文件名里的信息（地名等）始终可被历史索引搜到。

### 📦 含 v3.1.2 / v3.1.1 的全部修复
- 大图 / HEIC 照片正常出缩略图与 AI 检测（解除像素上限 + 内置 pillow-heif）
- 扫描跳过并清理 macOS `._xxx` 伴随文件
- 画廊解除 200 张上限、渐进式渲染
- 分析批次异常隔离、进度条修复、字段拖移修复
- 顶部徽章自动显示真实版本号

## 📦 下载与安装

提供 **macOS（Apple Silicon）** 与 **Windows（x64）** 两个平台，按需下载。

### macOS（Apple Silicon M1/M2/M3/M4）
下载 `Photo-Aiid-system_3.1.3_aarch64.dmg`：

1. 双击 `.dmg`，将 `Photo-Aiid-system.app` 拖入「应用程序」文件夹
2. **首次打开会被 macOS 拦截**（提示“无法验证开发者”或“已损坏，无法打开”）——这是因为本应用未经苹果签名，**不是病毒**
3. 解决方法（任选一种）：
   - **右键点击 App 图标 → 打开 → 在弹窗中再次点击「打开」**（只需一次，之后可正常双击）
   - 或在终端执行：
     ```bash
     xattr -dr com.apple.quarantine "/Applications/Photo-Aiid-system.app"
     ```

### Windows（x64）
下载 `Photo-Aiid-system_3.1.3_x64-setup.exe`（NSIS 安装包，推荐）双击运行安装；另提供 `Photo-Aiid-system_3.1.3_x64_zh-CN.msi` 供企业部署。首次运行若被 SmartScreen 拦截，点「更多信息 → 仍要运行」即可（未签名所致，非病毒）。
